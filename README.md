### Clone and install dependencies

```sh
git clone https://github.com/Lormida/fp-ts-examples.git \
&& cd fp-ts-examples \
&& pnpm i
```

### Run file locally

1. Put needed code (function or module) in `main.ts` file
2. Then either use:

```sh
npm run build:watch && npm run start
```

or

```sh
npm run dev
```
